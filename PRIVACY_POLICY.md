# Privacy Policy

Effective date: 2026-05-28

PinIG Media Downloader is a browser extension for downloading visible photos and videos from Instagram and Pinterest pages.

## Data Collection

PinIG Media Downloader does not collect, sell, transmit, or share personal data.

## Local Processing

The extension scans the current Instagram or Pinterest page locally in your browser to identify visible media URLs. These URLs are used only when you click a download button.

## Downloads

When you start a download, the selected media URL is sent to the browser's built-in Downloads API so the browser can save the file. The extension does not send this information to any external server.

## Storage

The extension uses local browser storage only to remember interface preferences, such as the floating panel's position and size.

## Remote Code

The extension does not load or execute remote code.

## Contact

For questions, contact the extension maintainer.
